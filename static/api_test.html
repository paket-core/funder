1. register_user - registers a new user
  2. balance - user balance should be zero
  3. send_buls - send from owner to new user
  4. balance - note change in user's balance
  5. launch_package - launch a new package with that user in one of the roles. For the others use two of the three pre-defined and pre-funded auxiliary users built in to the system: launcher, courier, and recipient.
  6. balance - note change in launcher's balance
  7. accept_package - accept the package as courier
  8. balance - note change in courier's balance
  9. accept_package - accept the package as recipient
  10. balance - note change in courier's balance
